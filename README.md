# WhatsApp AI

App de chat com IA no estilo WhatsApp, similar a PolyBuzz.ai e Character AI.

## Estrutura

- `apps/web` — Frontend React + Vite + Tailwind
- `apps/api` — Backend FastAPI (Python)

## Stack

- **Frontend:** React, TypeScript, Tailwind CSS, Supabase Auth
- **Backend:** FastAPI, OpenAI API, PostgreSQL (Supabase)
- **Infra:** pnpm monorepo, Turbo, Vercel (front) + Render (back)
